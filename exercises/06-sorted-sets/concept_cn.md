# 练习 06：有序集合

## 你将学到

- 使用 `ZADD` 添加带分数的成员
- 使用 `ZRANGE` 和 `ZREVRANGE` 按排名获取
- 使用 `ZRANK` 和 `ZREVRANK` 获取排名
- 使用 `ZSCORE` 获取分数
- 使用 `ZCOUNT` 按分数范围计数
- 使用 `ZINCRBY` 递增分数
- 使用 `ZREM` 删除成员

## 为什么重要

有序集合是 Redis 最通用的数据结构。每个成员都有一个分数，成员始终按分数排序。可用于排行榜、优先队列、限流窗口、时间序列数据等。

## 常见陷阱

- **相同分数 → 字典序排列**：平局按字母顺序打破。
- **ZRANK 从零开始**：分数最低的成员排名为 0。
- **ZADD 会更新已有成员**：添加已存在的成员会更新其分数。
